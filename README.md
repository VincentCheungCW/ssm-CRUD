# ssm-CRUD
A RESTful WebService using Spring+SpringMVC+Mybatis, featuring CRUD

## 功能点
- 分页
- 数据校验
   - jQuery前端校验+JSR303后端校验
- ajax
- Restful URI
   - 使用HTTP协议请求方式的动词，来表示对资源的操作（GET（查询），POST（新增），PUT（修改），DELETE（删除））

## 技术点
- 基础框架SSM（Spring + SpringMVC + Mybatis）
- 数据库MySQL
- 前端框架Bootstrap
- 项目依赖管理Maven
- 分页PageHelper
- 逆向工程Mybatis Generator
